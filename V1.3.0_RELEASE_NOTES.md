# H5餐饮点餐系统 v1.3.0 发布说明

**发布日期**: 2026年3月27日  
**版本状态**: 正式发布

## 版本概述
v1.3.0版本专为小县城单店餐厅设计，添加了4个实用功能，保持系统轻量级和易用性。所有功能均经过完整测试，确保稳定性和用户体验。

## 新增功能

### 1. 营业时间设置
- **功能描述**: 支持设置多时段营业时间，自动控制顾客端点餐状态
- **使用场景**: 餐厅可以根据实际营业时间自动开启/关闭点餐功能
- **管理方式**: 管理后台"系统设置"页面进行配置

### 2. 库存提醒
- **功能描述**: 菜品库存管理，售罄时自动显示售罄状态
- **使用场景**: 避免顾客下单已售罄菜品，提升用户体验
- **管理方式**: 管理后台菜品编辑页面设置库存数量

### 3. 基础营销功能
- **功能描述**: 支持优惠券和满减活动管理
- **使用场景**: 促进销售，增加客单价
- **功能特性**:
  - 优惠券创建、验证、状态管理
  - 满减活动配置
  - 营销活动启用/禁用控制

### 4. 简单数据导出
- **功能描述**: 订单数据CSV格式导出
- **使用场景**: 数据备份、财务统计、经营分析
- **功能特性**:
  - 支持按日期范围筛选
  - 支持按订单状态筛选
  - 自动CSV格式处理，兼容Excel

## 技术改进
